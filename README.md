# Zomato Analysis Dashboard

## Overview
This project presents an interactive dashboard developed using Excel/Power BI to analyze restaurant data from Zomato. The objective is to transform raw data into meaningful insights that help understand customer behavior, pricing trends, and restaurant distribution across different countries.

--

## Tools & Technologies
- **Power BI**
- **Excel**
- **SQL (MySQL)**
- **Tableau**
- **DAX**
- **Power Query**
- **Data Modelling**

## Key Highlights
- Total Restaurants: **9,551**
- Total Sales: **11 Million**
- Total Votes: **1 Million**
- Countries Covered: **15**
- Average Rating: **2.89**

---

## Dashboard Features

### Overall Metrics
Displays key performance indicators including total restaurants, sales, votes, countries, and average ratings.

### Home Delivery Analysis
Compares the number of restaurants offering home delivery versus those that do not.

### Country-wise Restaurant Distribution
Visual representation of restaurant counts across different countries.

### Price Category Analysis
Restaurants are categorized into:
- Low
- Medium
- High
- Premium

### Rating Distribution
Shows how restaurants are distributed across different rating levels.

### Trend Analysis
Tracks the growth of restaurants over time.

### Filters
- Year filter  
- Restaurant name filter

## Key Insights
- Most restaurants fall under the **low-price category**
- A majority of restaurants **do not offer home delivery**
- India has the highest number of restaurants
- Ratings are mostly concentrated in the mid-range (2.5–3.5)
